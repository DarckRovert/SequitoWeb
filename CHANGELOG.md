# Historial de Cambios (Changelog)

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto se adhiere a la arquitectura de versiones de **DarckRovert**.

## [5.0.0] - 2026-04-09
### Añadido
- Rediseño total de la interfaz con estética **Cyber-Void**.
- Nuevo motor de partículas en Canvas para efectos atmosféricos.
- Sección de Lore expandida con 5 actos canónicos.
- Galería de miembros del Séquito.
- Documentación institucional Diamond-Tier (Wiki, GitHub Templates).

### Corregido
- Error de opacidad en el canvas principal que ocultaba partículas.
- Fallo en el scroll suave en dispositivos móviles.

## [4.0.0] - 2025-12-20
### Añadido
- Integración con el Launcher de Turtle WoW.
- Sistema de copiado de URLs de repositorios.
