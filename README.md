## 🖤 ¿Qué es esto?

Este repositorio contiene el código fuente **Front-End** de la página de aterrizaje oficial de **El Séquito del Terror**. Construida puramente con HTML5, CSS3, e interacciones síncronas en JS Vainilla en un diseño *Cyber-Void* brutalista, actuando como portal de difusión principal y manual de instalación táctico para el ecosistema bélico del clan.

## ⚙️ Arquitectura Web

- **`index.html`**: El core. Un *One-Pager* inmersivo dividido estratégicamente en Lore, Arsenal (Ecosistema), Tutorial de Instalación con "1-Click-Copy" de 10 addons, y Reclutamiento.
- **`styles.css`**: Sistema de diseño basado en CSS Variables (`--fel-green`, `--void-purple`, `--horde-red`) inyectando estética inmersiva a través de mixins de *Glassmorfismo*, animaciones `clip-path` destructivas en los botones y glow estático interrelacionado.
- **`script.js`**: Controlador de DOM asíncrono. Gestiona el Canvas background de "cenizas del vacío", observadores de intersección nativos para animación al hacer scroll (`.reveal`) y lógica *responsive* de navegación.
- **`assets/`**: Archivos estáticos. Gráficos, el portal, y las ilustraciones de *Elnazzareno*.

## 🚀 Integración "Launcher-Ready"

La web incluye scripts custom inline de manipulación en el portapapeles (`copyToClipboard`) bajo la sección de instalación. Esto permite copiar instantáneamente las URLs de los 10 repositorios oficiales (desde `WCS_Brain` v9.3.1 hasta `BigWigs`) y pegarlas de golpe en la caja de descarga automatizada del Turtle WoW Addon Launcher.

## ⚔️ Despliegue Actual (v1.0)

La web ya consolida e instruye sobre la versión **v9.3.0 [God-Tier]** unificada a largo de todos los addons presentados en el **Arsenal**. Requerimientos de hosting: Cualquier servidor web estático (GitHub Pages, Netlify, Vercel) basta.
