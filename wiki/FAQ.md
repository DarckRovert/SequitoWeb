# FAQ - Preguntas Frecuentes

## General

### ¿Qué es El Séquito del Terror?
Es la hermandad de brujos y clases de élite más letal de Turtle WoW, operando bajo una filosofía de dominación táctica y uso intensivo de herramientas personalizadas.

### ¿Puedo usar los addons si no soy del clan?
Sí, el ecosistema de addons es Open Source bajo licencia MIT. Sin embargo, ciertas configuraciones y perfiles de **WCS_Brain** están optimizados para el uso interno del Séquito.

## Instalación

### El Launcher de Turtle WoW no reconoce la URL.
Asegúrate de copiar la URL completa. Si el Launcher falla, puedes realizar la instalación manual descargando el ZIP desde GitHub y extrayéndolo en la carpeta `Interface\AddOns\`.

### ¿En qué orden debo instalar los addons?
Es crítico instalar primero el **WCS_Brain**, ya que sirve como núcleo de sincronización para el resto de los módulos del arsenal.

### ¿Cómo abro el panel de control?
Usa el comando `/wcsbrain` o el alias corto `/wb` dentro del juego.

## Técnico

### El motor de partículas consume mucha CPU.
El motor está limitado a ~120 partículas para optimizar el rendimiento. En dispositivos de gama baja, las partículas se ajustan automáticamente según la resolución de pantalla.

### He encontrado un error visual.
Por favor, abre un [Issue en GitHub](https://github.com/DarckRovert/SequitoWeb/issues) utilizando la plantilla de Bug Report.
