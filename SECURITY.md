# Política de Seguridad

Este documento describe la política de seguridad para el portal oficial de **El Séquito del Terror**.

## Versiones Soportadas

Actualmente se proporciona soporte de seguridad para las siguientes versiones:

| Versión | Soportada |
| ------- | --------- |
| 5.x.x   | ✅         |
| < 5.0.0 | ❌         |

## Reporte de Vulnerabilidades

Si descubres una vulnerabilidad de seguridad en este portal, por favor envía un mensaje directo a **DarckRovert** a través de nuestro [Servidor de Discord](https://discord.gg/SfY8vfFWTC).

Por favor, no abras un Issue público para reportar una vulnerabilidad de seguridad.

## Alcance

Esta política cubre únicamente el código fuente del portal web alojado en este repositorio. Los addons individuales del ecosistema tienen sus propias políticas de seguridad en sus respectivos repositorios.
